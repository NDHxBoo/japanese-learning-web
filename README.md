# Japanese Vocabulary Learning Website

## Mục tiêu
Xây dựng website học từ vựng tiếng Nhật bằng cách import dữ liệu từ file Excel.

Website hỗ trợ:
- Import file Excel `.xlsx`
- Hiển thị danh sách từ vựng
- Tìm kiếm từ vựng
- Lọc theo môn học, Unit, Session
- Học bằng flashcard
- Làm quiz trắc nghiệm
- Lưu dữ liệu bằng LocalStorage

## Công nghệ sử dụng
- HTML
- CSS
- JavaScript thuần
- SheetJS để đọc file Excel
- LocalStorage để lưu dữ liệu

## Đối tượng sử dụng
- Sinh viên học tiếng Nhật
- Người học giáo trình Dekiru
- Người ôn JLPT N5/N4/N3

## Cách chạy
1. Mở project bằng VS Code
2. Cài extension Live Server
3. Click chuột phải vào `index.html`
4. Chọn `Open with Live Server`
5. Import file Excel từ vựng và bắt đầu học
